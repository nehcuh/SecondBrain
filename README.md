# Usage
1. Download this repo to your local disk e.g. "~/Documents/MySecondBrain"
2. Install Obsidian
3. Use Obsidian to open a local vault and specify to target repo, e.g. "~/Documents/MySecondBrain"

# Causion
1. The "HomePage.md" is homepage
2. Both 'dataview' and 'tasks' plugins are enabled, any task that is done should both check the metadata 'done' and content checkbox
